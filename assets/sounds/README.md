# Sound Assets

Add your audio files here for the game. Recommended files:

## Required Sound Effects
- `pop.mp3` - Block placement sound (short pop/click)
- `blast.wav` - Line clear explosion sound
- `combo.mp3` - Combo achievement sound
- `game_over.mp3` - Game over sound
- `error.mp3` - Invalid move sound
- `refill.mp3` - Hand refill sound

## Background Music
- `bgm_loop.mp3` - Main game background music (looping)

## Recommended Free Sources
- **Freesound.org** - Free sound effects
- **OpenGameArt.org** - Free game assets
- **ZapSplat** - Free sound effects for games
- **JSFXR** - Generate retro 8-bit sounds online

## Format Recommendations
- Use MP3 or WAV format
- Keep files small (< 100KB for SFX, < 1MB for BGM)
- Use 44.1kHz sample rate
- Mono for sound effects, stereo for BGM
